Uma das vantagens do formato *nexus* é que é um formato em blocos. Este acima é o bloco que contem sua matriz (é chamado *data block*), e ele começa em 
```BEGIN DATA;``` e termina em ```END;```. Então, ao invés de escrevermos todos os comandos no console do MrBayes, os escreveremos num próximo bloco.

Primeiramente, abra o arquivo <a href="files/best_scheme.txt" download="best_scheme.txt">best_scheme.txt</a> que fizemos com o PartitionFinder2 na aula anterior. Qual foi mesmo nosso esquema de partições que mais se 
encaixa nos nossos dados? Quais os modelos de substituição nucleotídica que mais se encaixam em cada uma das partições? 

Vamos preparar nosso bloco com os comandos do MrBayes com base nos resultados do PF2. Num editor de texto, adicione o seguinte no final do arquivo 
**hylodidae.nex**:

```
begin mrbayes;

  outgroup Limnomedusa_macroglossa;
	charset h1 = 1-2475;
	charset coi_pos1 = 2476-3187\3;
	charset coi_pos2 = 2477-3187\3;
	charset coi_pos3 = 2478-3187\3;
	partition PartitionFinder = 4:h1, coi_pos1, coi_pos2, coi_pos3;
	set partition=PartitionFinder;
	lset applyto=(1) nst=6 rates=invgamma;
	lset applyto=(2) nst=6 rates=propinv;
	lset applyto=(3) nst=1 rates=propinv;
	lset applyto=(4) nst=6 rates=gamma;
	prset applyto=(all) ratepr=variable;
	unlink statefreq=(all) revmat=(all) shape=(all) pinvar=(all) tratio=(all);
```
Em vez de explicarmos cada um desses parâmetros aqui, porque não abrimos o MrBayes e deixamos ele nos dizer o que cada um significa? Digite o comando ```help``` 
seguido de um outro comando, como por exemplo:
```
MrBayes > help outgroup

---------------------------------------------------------------------------   
   Outgroup                                                                      
                                                                                 
   This command assigns a taxon to the outgroup. The correct usage is:           
                                                                                 
      outgroup <number>/<taxon name>                                             
                                                                                 
   For example, "outgroup 3" assigns the third taxon in the matrix to be       
   the outgroup. Similarly, "outgroup Homo_sapiens" assings the taxon          
   "Homo_sapiens" to be the outgroup (assuming that there is a taxon named     
   "Homo_sapiens" in the matrix). Only a single taxon can be assigned to       
   be the outgroup.                                                              
                                                                                 
   ---------------------------------------------------------------------------  
```
Caso você não tenha o MrBayes instalado e quiser fazê-lo, você pode baixá-lo [aqui](https://nbisweden.github.io/MrBayes/download.html). 
O programa está nos dizendo que o comando ```outgroup``` define uma raiz. Agora faça com os outros comandos.

Vamos adicionar o resto dos comandos e fechar o bloco:
```
	mcmc ngen=1000000 nruns=2 nchains=4 samplefreq=1000 printfreq=1000 relburnin=yes burninfrac=0.25;
	sump;
	sumt;
end;
```
Isso indica que vamos rodar duas corridas com quatro correntes cada uma de 1.000.000 de gerações, amostrando e *printando* na tela a cada 1.000 gerações, 
descartando os primeiros 25% das amostras como burnin. Este descarte ocorre porque no começo a busca pode estar em um lugar muito distante no espaço amostral da 
nossa resposta ótima, o que pode atrapalhar nosso resultado final. Então é melhor esperar as corridas se estabilizarem para começar a considerar os resultados.

Agora salve este novo arquivo como **hylodidae_mb.nex**.
